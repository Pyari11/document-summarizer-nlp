# NLP-Based Multi-Format Document Summarizer
This project is a Python-based document summarization tool that can summarize
PDF, DOCX, and TXT files.

## Features
- Supports PDF, DOCX, and TXT documents
- Automatic heading detection
- Bullet-point summaries
- Handles large documents with page limits
- Simple NLP-based extractive summarization

## Technologies Used
- Python
- NLTK
- PyPDF2
- python-docx

## How to Run
1. Install dependencies
pip install PyPDF2 python-docx nltk
2. Run the program
python summarizer.py
3. Enter the file path when prompted.
Example:
Enter file path (.pdf, .docx, .txt):
document.pdf
